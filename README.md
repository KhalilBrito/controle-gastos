# Controle de Gastos

Aplicação Full Stack desenvolvida para praticar ASP.NET Core Web API e React.

## Tecnologias

- ASP.NET Core 10
- Entity Framework Core
- SQLite
- React
- TypeScript
- Vite

## Funcionalidades

- Cadastro de pessoas
- Cadastro de receitas e despesas
- Exclusão de pessoas
- Dashboard com totais
- API REST

## Como executar

### Backend

cd Backend

dotnet restore

dotnet ef database update

dotnet run

### Frontend

cd Frontend

npm install

npm run dev

Frontend:

http://localhost:5173

Backend:

http://localhost:5076
